# Park Maps — drop the official PDFs here

The dashboard's 🗺️ "Park map" chips link to these exact filenames (place PDFs in this folder):

| File to save here | Park | Where to get it |
|---|---|---|
| magic-kingdom.pdf | Magic Kingdom | disneyworld.disney.go.com/maps → Magic Kingdom → print/save as PDF |
| epcot.pdf | EPCOT | same site |
| hollywood-studios.pdf | Hollywood Studios | same site |
| animal-kingdom.pdf | Animal Kingdom | same site |
| universal-studios.pdf | Universal Studios FL | universalorlando.com → Plan Your Visit → Maps (or the Universal app) |
| islands-of-adventure.pdf | Islands of Adventure | same |
| epic-universe.pdf | Epic Universe | same |
| volcano-bay.pdf | Volcano Bay | same |

Tips:
- Any PDF (even a full-page screenshot saved as PDF) works — the chip just opens the file.
- Publishing to GitHub Pages? Upload this maps/ folder too and the links keep working.
- The MDE and Universal Orlando apps have live maps — these PDFs are the offline backup.
